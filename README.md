# 瓜瓜记单词 🐸

一款专为小学生设计的英语单词学习微信小程序，让孩子爱上背单词！

## ✨ 功能特点

### 📚 首页
- 环形进度条显示今日学习进度
- 🔥 连续打卡天数展示
- 待复习单词数量角标提醒
- 快捷按钮：开始学习、快速复习、随机测试
- 按年级（三至六年级）选择学习内容

### 📖 单词学习
- 卡片式学习，正面英文+音标，背面中文+例句
- 点击卡片触发 3D 翻转动画
- 滑动手势：右滑"认识"、左滑"不认识"、上滑"收藏"
- 点击喇叭图标播放发音
- 学习结束显示本轮统计弹窗
- 基于 SM-2 算法自动安排复习计划

### 🎯 练习测试
- 三种题型：选择题 / 填空题 / 听力题
- 顶部倒计时进度条
- 答对绿色动画，答错红色抖动动画
- 练习结束页：正确率、用时、错题回顾

### 📊 学习进度
- 柱状图展示近 7/30 天学习趋势
- 日历热力图显示学习频率
- 饼图显示词汇掌握度分布
- 成就徽章墙：首次学习、连续7天、掌握100词等

### ⚙️ 设置
- 每日学习目标滑块（10-100个）
- 学习提醒开关 + 时间选择
- 音效 / 震动反馈 / 自动发音开关
- 外观模式：跟随系统 / 浅色 / 深色
- 重置学习进度（二次确认）

## 🚀 运行说明

1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 打开微信开发者工具，选择"导入项目"
3. 选择本项目文件夹
4. 填写 AppID（可使用测试号）
5. 点击"编译"即可预览

## 📱 预览

在微信开发者工具中编译后，可以在模拟器中预览，或扫码在真机上测试。

## 📝 单词数据

目前包含 200+ 小学英语单词，按年级分类：
- 三年级：50 个基础单词
- 四年级：50 个进阶单词
- 五年级：50 个中级单词
- 六年级：50 个高级单词

每个单词包含：英文、音标、中文释义、例句

## 🔧 技术栈

- 微信小程序原生开发
- SM-2 间隔重复算法
- Canvas 环形进度条
- CSS 3D 翻转动画
- 本地存储 (wx.Storage)
